Dublin Delights is a website which gets flight information from Dublin Airport
When not logged in:
      Home:
          No Input: Gets 15 alphabetical destination flights from Dublin
          Destination: Gets flights available soon from the destination from Dublin
          Date: Gets the available flights leaving on that date
          Destination + Date: Gets the flights available for that date and Destination
          If No flight available: Just gives message saying so.
When Logged In:
You now have access to the Logout.py, remove.py, userpage.py and Login.py and register.py give "Already logged in"
    Home:
        Gets all same for each input but you can now save any information to your account
    Userpage:
        Your Saved flights show here
    Remove:
        You can remove saved flights here.
